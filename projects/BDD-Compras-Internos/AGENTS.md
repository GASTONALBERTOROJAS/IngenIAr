# BDD-Compras-Internos — IngenIAr Project

This project follows the IngenIAr system contract. See root `AGENTS.md` for the full workflow.

## Required workflow

For every meaningful request:

1. interpret request
2. structure requirements
3. define architecture or approach
4. create an execution plan
5. implement
6. validate with QA
7. close only if QA = OK

## Rules

- keep changes inside scope
- record plans in `.ai-dev/plans/` and QA evidence in `.ai-dev/qa/`
- do not close work without QA PASS
